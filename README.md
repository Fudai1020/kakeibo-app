## サービスへの想い
私は現在、パートナー遠距離恋愛をしておりその中でパートナーとの将来のことをきちんと考えていることを示すため、
また自分自身家計簿をつける習慣がなく、貯金が苦手だったので、離れた相手でも一緒に家計簿を共有できる家計簿アプリ
があったらいいなと思い作りました。
